# 🏦 Loan Analytics Dashboard

An interactive Power BI dashboard developed to analyze a financial institution's loan portfolio from the perspectives of loan exposure, borrower credit quality, repayment burden, and overall credit risk.

The dashboard enables stakeholders to identify high-risk borrower segments, understand lending patterns, and monitor portfolio health through interactive visualizations.

---

# 📌 Project Objective

The objective of this project is to transform raw loan data into meaningful business insights using Power BI.

This dashboard helps answer questions such as:

- Which loan purposes contribute the highest loan exposure?
- Which customer segments carry higher financial risk?
- How are loans distributed across income groups and home ownership categories?
- How do monthly debt, annual income, and credit exposure relate?
- Which borrowers exhibit signs of higher delinquency, bankruptcy, or tax liens?
